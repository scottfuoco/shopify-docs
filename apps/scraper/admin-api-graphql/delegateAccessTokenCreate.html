delegateAccessTokenCreate - admin-graphql - MUTATION

Version: 2025-01

Description:

Creates a delegate access token. To learn more about creating delegate access tokens, refer to [Delegate OAuth access tokens to subsystems](https://shopify.dev/docs/apps/build/authentication-authorization/access-tokens/use-delegate-tokens).

Access Scopes

Arguments

  • input: DelegateAccessTokenInput! - The input fields for creating a delegate access token.

Returns

  • delegateAccessToken: DelegateAccessToken The delegate access token.
  • shop: Shop! The user's shop.
  • userErrors: DelegateAccessTokenCreateUserError! The list of errors that occurred from executing the mutation.
